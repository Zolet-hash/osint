- Netcat: network swiss army knife
- ```shell
  nc -nv 127.0.0.1 80
  -n --> numeriv only ip addr no DNS
  -v -> verbous
  -w secs timeouts for connects and final net
  -u -> UDP mode
  
  nc -nvu 127.0.0.1 443
  ```
-
- netcat script
- ```shell
  # Timeout control for responsive scanning
  nc -nv -w 3 target-ip 80
  
  # Banner grabbing with input
  echo -e "GET / http/1.1\r\n:Host target-domain.com\r\n\r\n" | nc -nv - target-ip 80
  ```
