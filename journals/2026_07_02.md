- Netcat: network swiss army knife
- ```shell
  nc -nv 127.0.0.1 80
  -n --> numeriv only ip addr no DNS
  -v -> verbous
  -w secs timeouts for connects and final net
  -u -> UDP mode
  
  nc -nvu 127.0.0.1 443
  ```
-
- Advanced Netcat Techniques
- ```shell
  # Timeout control for responsive scanning
  nc -nv -w 3 target-ip 80
  
  # Banner grabbing with input
  echo -e "GET / HTTP/1.1\r\nHost: target-domain.com\r\n\r\n" | nc -nv target-ip 80
  
  # Multiple port grabber
  for port in 21 22 23 25 53 80 110 143 443 993 995; do
  	echo "==port $port =="
      nc -nv -w 2 target-ip 80
  done
  
  # Save banner info to a file
  nc -nv target-ip 80 > http_banner.txt
  nc -nv target-ip 22 > ssh_banner.txt
  ```
